# OpenPalette Support

## Where to get help

- **Documentation** — See [docs/](docs/) for architecture, design system, and product spec.
- **Bug reports** — Open a [GitHub Issue](https://github.com/sparshsam/openpalette/issues/new).
- **Feature requests** — Open a [GitHub Issue](https://github.com/sparshsam/openpalette/issues/new) with the `enhancement` label.
- **Discussions** — Use [GitHub Discussions](https://github.com/sparshsam/openpalette/discussions) for questions.

## Before opening an issue

1. Check [existing issues](https://github.com/sparshsam/openpalette/issues) for duplicates.
2. Search [docs/](docs/) for relevant guidance.

## Response times

This is a side project. Issues are typically addressed within a week.
